Pushover client by @macwinnie

This is the documentation overview for Pushover client by @macwinnie.

To install this package, please run:

            pip install macwinnie_py_pushover_client

Usage could then e.g. be: 

            from macwinnie_py_pushover_client import Pushover

            po = Pushover.pushover()
            po.setDB()

            po.messageAll( 'This is a demonstration message.' )

You need to define some environmental variables. That could be within a .env file and load by python-dotenv: 

            PUSHOVER_APP=<app_token_from_pushover_website>
            PUSHOVER_USERS=["<user_token_1>","<user_token_2>"]
